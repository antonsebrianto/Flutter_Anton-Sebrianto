# Resume Dart Object Oriented Programming

3 Poin yang dipelajari dari materi Dart Object Oriented Programming

- Dengan menggunakan OOP, kita dapat membuat suatu program dengan struktur codingan yang lebih tertata dan memudahkan kita untuk melakukan troubleshooting ketika terjadi error pada program. Dan dengan adanya OOP kita tidak perlu meng-inialisasi berulang kali suatu fungsi, karena kita dapat menggunakan fungsi tersebut berulang kali sesuai dengan keinginan kita.
- Variabel dan Property tergolong sama. Yang menjadi pembeda antara variabel dan property itu hanya penempatannya, untuk property itu terletak di dalam sebuah class. Dan untuk menggunakan property seperti memanggil sebuah object.
- Fungsi dan Method juga tergolong sama. Hanya saja method terletak di dalam sebuah class, dan untuk dapat mengakses sebuah method dapat dipanggil seperti memanggil sebuah object.
